# mas-app 
# running example, Windows environment only.
A no-frills command system, you can "capture screen, send email, file checking, export stuff to PDF and so on"

干净利落，没有过多修饰的命令行系统。

问：no-frills是什么意思？

答：adj. 不提供不必要服务的； 不加装璜；

$$> git clone git@github.com:acesfullmike/masrun.git

1) Make sure your JRE is available, set the right value in mas.bat 'set JRE_HOME'.
   
   PS: The version for JRE should be 8.0 and higher, where to download?
   
   JRE: http://download.oracle.com/otn-pub/java/jdk/8u71-b15/jre-8u71-windows-x64.exe
   
2) Double click nomad.bat

3) Type ? or ' for realtime help
  
4) For online help, check https://raw.githubusercontent.com/acesfullmike/masrun/master/help.txt
